# swaggerfile
We provide a swaggerfile in OpenAPI Specification 2 for creating automatically webdataforge api clients in many languages. 
One way to quickly create an API client is to use [Swagger Codegen](https://swagger.io/docs/open-source-tools/swagger-codegen/ "Swagger Codegen").

## webdataforge backend
Please use the following URL as service root URL to our REST interface: https://backend.webdataforge.com. If you want to try out our REST API interactively,
please visit https://backend.webdataforge.com/swagger-ui.html.
